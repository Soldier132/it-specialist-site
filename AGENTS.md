# Project instructions (source of truth)

You are working on a turnkey website for an IT specialist.

## Read first (requirements)
- /docs/spec/tech-spec.md
- /docs/spec/acceptance.md
- /docs/spec/sitemap.md
- /docs/spec/seo.md
- /docs/spec/integrations.md

## Non-negotiables
- Pages: Home, About, Services, Portfolio, Reviews, Contacts (+ optional Blog later)
- Fully responsive layout
- Forms:
  - feedback: name, email, message
  - service request: name, phone, email, request description
  - optional newsletter: email
- Form delivery: send to configured email (SMTP preferred) and/or store in admin
- SEO basics: clean URLs, per-page title/meta, correct H1-H3 hierarchy, image alt, sitemap.xml, robots.txt
- Integrations: Telegram/WhatsApp buttons, mailto links, Google Analytics, Search Console readiness, Google Maps embed, social links

## Working rules
- Implement in small verifiable steps.
- Keep performance and security as priorities.
- If something is ambiguous: make a reasonable default and record it in tech-spec.md under "Assumptions".
