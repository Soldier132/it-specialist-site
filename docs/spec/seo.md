# SEO Requirements

- Clean human-readable URLs (e.g. /services/server-setup)
- Per-page configurable: <title>, meta description
- Headings: one H1 per page, correct H2/H3 hierarchy
- Images must have alt attributes
- Generate sitemap.xml and robots.txt
- Canonical URLs and redirects (www <-> non-www)
- Basic performance: compressed images, caching where appropriate
